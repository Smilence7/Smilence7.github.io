    IDEA Keymap

    Posted on 2021-01-31

    本文归纳自官方文档与 IntelliJ-IDEA-Tutorials

    for Windows

    Ctrl

    快捷键 描述
    Ctrl + F 在当前文件进行文本查找
    Ctrl + R 在当前文件进行文本替换
    Ctrl + Z 撤销
    Ctrl + C 复制光标所在行 或 复制选择内容
    Ctrl + X 剪切光标所在行 或 剪切选择内容
    Ctrl + D 复制光标所在行 或 复制选择内容，并把复制内容插入光标位置下面
    Ctrl + Y 删除光标所在行 或 删除选中的行
    Ctrl + W 递进式选择代码块。可选中光标所在的单词或段落，连续按会在原有选中的基础上再扩展选中范围
    Ctrl + E 显示最近打开的文件记录列表
    Ctrl + N 根据输入的类名查找类文件
    Ctrl + J 插入自定义动态代码模板
    Ctrl + P 方法参数提示显示
    Ctrl + U 前往当前光标所在的方法的父类的方法 / 接口定义
    Ctrl + B 进入光标所在的方法/变量的接口或是定义处，等效于 Ctrl + 左键单击
    Ctrl + / 注释光标所在行代码，会根据当前不同文件类型使用不同的注释符号
    Ctrl + F1 在光标所在的错误代码处显示错误信息
    Ctrl + F3 调转到所选中的词的下一个引用位置
    Ctrl + F4 关闭当前编辑的文件
    Ctrl + F9 执行 Make Project 操作
    Ctrl + F11 选中文件 / 文件夹，使用助记符设定 / 取消书签
    Ctrl + F12 弹出当前文件结构层，用来在当前文件中定位
    Ctrl + Space 基础代码补全，默认在 Windows 系统上被输入法占用，需要进行修改，建议修改为 Ctrl + 逗号
    Ctrl + Delete 删除光标后面的单词或是中文句
    Ctrl + BackSpace 删除光标前面的单词或是中文句
    Ctrl + 1,2,3...9 定位到对应数值的书签位置
    Ctrl + 左键单击 在打开的文件标题上，弹出该文件路径
    Ctrl + 光标定位 按 Ctrl 不要松开，会显示光标所在的类信息摘要
    Ctrl + ← 光标跳转到当前单词 / 中文句的左侧开头位置
    Ctrl + → 光标跳转到当前单词 / 中文句的右侧开头位置
    Ctrl + O 选择可重写的方法
    Ctrl + I 选择可继承的方法
    Ctrl + + 展开代码
    Ctrl + - 折叠代码
    Ctrl + H 显示当前类的层次结构
    Ctrl + T 版本控制更新项目
    Ctrl + K 版本控制提交项目

    Alt

    快捷键 描述
    Alt + ` 显示版本控制常用操作菜单弹出层
    Alt + Q 弹出一个提示，显示当前类的声明 / 上下文信息
    Alt + F1 显示当前文件选择目标弹出层，弹出层中有很多目标可以进行选择
    Alt + Enter IntelliJ IDEA 根据光标所在问题，提供快速修复选择，光标放在的位置不同提示的结果也不同
    Alt + Insert 代码自动生成，如生成对象的 set / get 方法，构造函数，toString() 等
    Alt + ←/→ 切换当前已打开的窗口中的子视图
    Alt + ↑/↓ 当前光标跳转到当前文件的前一个/后一个方法名位置
    Alt + 1,2,3...9 打开/关闭对应视图（1是project）

    Shift

    快捷键 描述
    Shift + F2 跳转到上一个高亮错误 或 警告位置
    Shift + F3 在查找模式下，查找匹配上一个
    Shift + F7 在 Debug 模式下，智能步入。断点所在行上有多个方法调用，会弹出进入哪个方法
    Shift + F8 在 Debug 模式下，跳出，表现出来的效果跟 F9 一样
    Shift + F9 等效于点击工具栏的 Debug 按钮
    Shift + F10 等效于点击工具栏的 Run 按钮
    Shift + F11 弹出书签显示层
    Shift + End 选中光标到当前行尾位置
    Shift + Home 选中光标到当前行头位置
    Shift + 左键单击 点击文件名快速关闭文件
    Shift + 滚轮 横向滚动文件

    for Mac

    Editing

    快捷键 描述
    ⌃ + Space 基本的代码补全（补全任何类、方法、变量）
    ⌃ + Shift + Space 智能代码补全（过滤器方法列表和变量的预期类型）
    ⌃ + Shift + Enter 自动结束代码，行末自动添加分号
    ⌘ + P 显示方法的参数信息
    ⌃ + J 显示当前位置的变量、方法的 Documentation 内容
    Shift + F1 查看外部文档
    ⌘ + 光标停留 显示代码简要信息
    ⌘ + F1 在错误或警告处显示具体描述信息
    ⌘ + N 生成代码
    ⌃ + O 重写父类方法
    ⌃ + I 显示代码简要信息
    ⌘ + ⌥ + T 生成包围代码
    ⌘ + / 注释/取消注释与行注释
    ⌘ + ⌥ + / 注释/取消注释与块注释
    ⌥ + ↑/↓ 跳转代码块
    ⌃ + Shift + Q 显示上下文信息
    ⌥ + Enter 显示意向动作和快速修复代码
    ⌘ + ⌥ + L 格式化代码
    ⌃ + ⌥ + O 优化import
    ⌘ + Shift + V 从缓冲区中选择粘贴
    ⌘ + Shift + U 切换大小写
    ⌃ + Shift + J 智能的将代码拼接成一行
    ⌘ + Enter 智能的拆分拼接的行
    ⌘ + -/= 展开/折叠代码块

    Search/Replace

    快捷键 描述
    ⌘ + G 查找模式下，向下查找
    ⌘ + Shift + G 查找模式下，向上查找
    ⌘ + R 文件内替换
    ⌘ + Shift + F 全局查找
    ⌘ + Shift + R 全局替换
    ⌥ + F7 find usage
    ⌘ + ⌥ + F7 show usage

    Run/Debug

    快捷键 描述
    ⌃ + ⌥ + R 弹出 Run 的可选择菜单
    ⌃ + ⌥ + D 弹出 Debug 的可选择菜单
    F7 步入(step into)
    F8 步过(step over)
    Shift + F7 智能步入
    Shift + F8 跳出
    ⌘ + ⌥ + R 继续运行
    ⌘ + Shift + F8 查看所有断点

    Navigation

    快捷键 描述
    ⌘ + B 进入光标所在的方法/变量的接口或是定义处
    ⌃ + H 显示当前类的层次结构
    ⌘ + Shift + H 显示当前类的层次结构
